# Museum Dioramatum Visionum
This is a game developed for Hack Club Horizons, or, to be more exact, a prototype of one. 
You play as a visitor of a mysterious museum with multiple diorama like platformer levels to complete

## HOW THIS WAS MADE
This was made with Godot, Blender, Yoink, Aseprite and Ms Paint.
## How to run
1. Download the zip file here or on itch: https://yatagarasu-the-17th.itch.io/pinacotheca-visionis
2. Extract
3. run the .exe file (you might need to turn off windows defender)
4. Don't worry if it takes a while to load, i made a ton of assets and set up nicer lighting than usual for my games.
5. Enjoy!
Compatible with Windows, 2 graphics presets included
